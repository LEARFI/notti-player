## ✨ UI/UX Особенности

Главная фишка Notti Player — **тактильный дизайн кнопок**. Мы ушли от статики к живым микро-взаимодействиям.

### 🕹 Анимированное управление
*   **Active Scaling:** Кнопки физически реагируют на нажатие (уменьшение масштаба), создавая ощущение работы с реальным устройством.
*   **Glassmorphism Control:** Панель управления использует эффект матового стекла (`backdrop-filter`), адаптируясь под фон обложки трека.
*   **Pulse Playback:** Центральная кнопка Play имеет мягкую цикличную анимацию свечения во время паузы, подсказывая следующее действие.
*   **Smooth Progress:** Индикатор прогресса плавно следует за треком без рывков.

## 💻 Пример реализации анимации кнопок

Для создания "живых" кнопок в проекте используется следующий подход (React + CSS):

```css
/* Эффект нажатия (Glass Press) */
.notti-btn-active {
  transition: transform 0.2s cubic-bezier(0.4, 0, 0.2, 1);
}

.notti-btn-active:active {
  transform: scale(0.9); /* Кнопка «вдавливается» */
  filter: brightness(1.2);
}
